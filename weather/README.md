## Packages Installation

1. Open a console
2. Go to the directory `/weather`
3. Follow those command lines :

```sh
❯ npm install public-ip
❯ npm install chalk
❯ npm install cli-table
❯ npm install minimist
```
![](Screen1.png)

4. Test
```sh
❯ node index.js
```
