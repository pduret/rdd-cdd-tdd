## Packages Installation

- Open a console
- Go to the directory `/weather`
- Follow those command lines :

```sh
❯ npm install public-ip
❯ npm install chalk
❯ npm install cli-table
❯ npm install minimist
```

- Test
```sh
❯ node index.js
```
